Responsive Fragments

Load in additional HTML fragments based on device category 'keyword', set by CSS3 MediaQuery viewport width.

Change the viewport of your browser, starting really small (smartphone small), and going wider, more HTML content is loaded in using jQuery Ajax based on HTML5 data-attribute URL on div-tag. Making the web browser window smaller hides the (already loaded) content again.
